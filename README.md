# TrainRadar 🚆

A web application that displays live UK train locations 
on an interactive map.

Built as part of my MSc dissertation at the 
University of Liverpool (COMP702, 2025/26).

## What it does
- Shows live UK train positions on an interactive map
- Lets users search for and select individual trains
- Estimates where a train currently is between stations
- Shows last departure time and predicted next arrival time
- Displays journey completion percentage

## Tech Stack
- **Backend:** Python 3, Flask
- **Frontend:** HTML5, CSS3, JavaScript, Leaflet.js
- **Data:** Real Time Trains API
- **Map:** OpenStreetMap
- **Deployment:** PythonAnywhere
